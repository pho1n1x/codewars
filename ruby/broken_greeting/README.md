# Kata
Name: [Broken Greetings](https://www.codewars.com/kata/50654ddff44f800200000001)

Difficulty: **8 Kyu**

# Description
Correct this code, so that the greet function returns the expected value.

# Scenario
```ruby
class Person
  def initialize(name)
    @name = name
  end
  
  def greet(other_name)
    "Hi #{other_name}, my name is #{name}"
  end
end
```
