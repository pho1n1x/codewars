Difficulty: **8 Kyu**

# Description
Create a function called `shortcut` to remove all the **lowercase** vowels in a given string. Don't worry about uppercase vowels.

# Example
```ruby
shortcut("codewars") # --> cdwrs
shortcut("goodbye")  # --> gdby
```
